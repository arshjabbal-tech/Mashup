# 🎵 Mashup Generator

A command-line Python program that creates an audio mashup from multiple videos.

## 📌 Features

- Convert videos into audio files
- Trim first Y seconds from each audio
- Merge all trimmed audio into a single mashup
- Command-line based execution
- Uses PyPI libraries (moviepy, pydub)

---

## ⚙️ Requirements

- Python 3.11
- ffmpeg installed
  ## 📦 PyPI Packages Used
- moviepy
- pydub

Install dependencies:

pip install moviepy==1.0.3 pydub

---

## 🚀 Usage

Run from terminal:

python 102303561.py "SingerName" NumberOfVideos AudioDuration OutputFileName

Example:

python 102303561.py "Sharry Maan" 20 25 output.mp3
